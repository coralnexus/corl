    ## Welcome to CORL (Coral Orchestration and Research Library)

    CORL is a cloud application framework and command line utility that provides easy and extensible creation and management of project based networks and nodes and relevant infrastructure.

    The CORL system is built on top of our general purpose Nucleon application framework.

    Contributing to CORL

    • Check out the latest {major}.{minor} branch to make sure the feature hasn't been implemented or the bug hasn't been fixed yet.

    • Check out the issue tracker to make sure someone already hasn't requested it and/or contributed it.

    • Fork the project.

    • Start a feature/bugfix branch.

    • Commit and push until you are happy with your contribution.

    • If possible, add tests for it. This is important so I don't break it in a future version unintentionally.

    • Please try not to mess with the Rakefile, version, or history. If you want to have your own version, or is otherwise necessary, that is fine, but please isolate to its own commit so I can cherry-pick around it.
